\## 项目简介

基于AI工具协作完成的八皇后问题回溯求解器，包含完整的单元测试、Bug引入与AI修复流程。



\## 环境要求

\- Python 3.8+

\- pytest 7.0+

\## 实现思路

使用回溯法逐行放置皇后，通过集合记录已占用的列和两条对角线（行-列、行+列），剪枝无效路径，最终返回所有合法解。



\## 测试覆盖

\- n=1：验证单皇后场景（解数1）

\- n=2：验证无解场景（解数0）

\- n=4：验证小棋盘合法解（解数2）

\- n=8：验证标准八皇后解数（解数92）



\## Bug修复说明

故意修改核心判断逻辑（将`not in`改为`in`）导致测试失败，AI快速定位逻辑错误并修复，验证了AI在代码调试中的有效性。
